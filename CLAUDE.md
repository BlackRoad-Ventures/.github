3. Preserve the centered footer attribution

## AI Assistant Guidelines

### When Working on This Repository

1. **Profile Updates**: Changes to `profile/README.md` affect the public organization page - review carefully
2. **Link Validation**: Verify all external links are correct and accessible
3. **Ecosystem Consistency**: Maintain alignment with other BlackRoad organization profiles
4. **Minimal Changes**: Only modify what's necessary - avoid reformatting unchanged content

### Common Tasks

| Task | Action |
|------|--------|
| Update organization profile | Edit `profile/README.md` |
| Add issue template | Create `.github/ISSUE_TEMPLATE/<name>.md` |
| Add PR template | Create `.github/PULL_REQUEST_TEMPLATE.md` |
| Update this guide | Edit `CLAUDE.md` |

### What NOT to Do

- Do not remove the ecosystem attribution footer
- Do not break links to sister organizations
- Do not add sensitive or private information (this is a public repo)
- Do not dramatically change the established formatting style
